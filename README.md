# Mision TI | Proyecto para el Curso de React JS de Coder House

Mi proyecto es una tienda online de productos electronicos desarrollada con React JS.

## Tecnologías usadas

- Visual Studio Code
- ReactJS
- Javascript(ES6)
- Html5
- Scss
- Flexbox y Grid
- npm
- Mui
- Firebase
- Git
